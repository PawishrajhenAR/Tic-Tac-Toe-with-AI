# Tic-Tac-Toe with AI

A simple and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**. Play with a friend or challenge an AI opponent in a clean and responsive interface.

---

## Features

- **Play with Friend Mode** – Local two-player gameplay.
- **Play with AI Mode** – Challenge a basic AI opponent.
- **Responsive Design** – Works across desktop and mobile.
- **Enhanced UI** – Smooth styling and animations.
- **Win Detection & Reset** – Automatically detects wins, draws, and supports quick restart.

---

## Technologies Used

- **HTML** – Game structure  
- **CSS** – Visual design and layout  
- **JavaScript** – Gameplay mechanics, AI logic, and event handling  

---

## How to Play

1. Click **Play with Friend** for a two-player game.  
2. Click **Play with AI** to compete against the computer.  
3. Players alternate placing X and O on the board.  
4. The game displays the winner or a draw.  
5. Use **Reset** to start a new match.
